## Zastosowane wzorce projektowe:

### Wzorce konstrukcyjne :

- Singleton

- Budowniczy

### Wzorce Strukturalne:

- Dekorator

- Fasada

### Wzorce Czynnościowe:

- Obserwator

- Iterator
